## What is Color Blooms?
Color blooms is an interactive creative coding project based on playing
with colors, shapes and randomization. The user can play with the provided parameters
to create a blooming field of colors.

## What does this repository contain?
This repository contains the source code for the project. The project is hosted
on GitHub pages for now.
